Positioning

Push and pull

File location: sass/layout/helpers/_adjustment.scss

Push and pull classes are used to fine tune the position of html elements.

We use two methods: push and pull classes, or translate-push and translate-pull classes. Pushing/pulling will also influence sibling element(s), while translate push/pull will move the element independently of its surroundings.

Class Name Comment
.push1v (...) .push20v Push 1-20px vertically (down)
.pull1v (...) .pull20v Pull 1-20px vertically (up)
.push1h (...) .push20h Push 1-20px horizontally (right)
.pull1h (...) pull20h Pull 1-20px horizontally (left)
.translate-push1v (...) -push20v Move 1-20px down
.translate-pull1v (...) -pull20v Move 1-20px up
.translate-push1h (...) -push20h Move 1-20px right
.translate-pull1h (...) -pull20h Move 1-20px left
