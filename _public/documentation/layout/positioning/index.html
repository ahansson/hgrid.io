Positioning

Push and pull

File location: sass/layout/helpers/_adjustment.scss

Push and pull classes can be used to fine tune the position of html elements. We use two methods: margin or transform. The first will also affect sibling element(s), while the latter will move the element independently of other elements.

No push

Push right →

No push

.translate-push10h

Pull left ←

No transform

.translate-pull20h

Pull up ↑

No pull

.translate-pull20v
