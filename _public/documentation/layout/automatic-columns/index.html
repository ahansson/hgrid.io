Automatic Content Columns

In order to quickly mark up a flexible layout grid, start by adding the .row class to an HTML element. This creates a flex container with special configuration for the column system to fit in.

File location: sass/layout/_auto_columns.scss

Any number of elements with the class .column inside the .row will split its width equally: 

<div class="container-max">
  <div class="row">
    <div class="column">
      <p>Column 1 (auto)</p>
    </div>
    <div class="column">
      <p>Column 2 (auto)</p>
    </div>
    <div class="column">
      <p>Column 3 (auto)</p>
    </div>
  </div>
</div>

The Sass variable $column-gutter-width controls the distance between columns. You can also overwrite the regular CSS for .column and modify the default value of 0.8% on each side.

The number of columns in each .row determines the width of each column.

Columns will by default stack on viewports that are less than 769px wide. Breakpoints for responsiveness can be adjusted in _variables.scss or overwritten with your own values when you import hgrid as either CSS or Sass.

The default breakpoint where columns stack can be overridden by modifying hgrid's $stack-point variable (read more here). You can also override it by adding an alternative breakpoint with $custom-stack-point and the class .custom-stack after the .row class.

Use .no-stack on the element that holds the .row class in order to prevent stacking.

Note: Columns would normally stack vertically on handheld devices (< 769px wide). For the sake of the examples on this page we have applied .no-stack after .row so they make sense for readers on any device.

Containers can be nested inside each other as long as you wrap each new level in a new .row. 

<div class="row">
  <div class="column border-medium padding-5 radius-8"></div>
  <div class="column border-medium padding-5 radius-8">
    <div class="row margin-bottom-0">
      <div class="column padding-10 bg-gray-light text-center">.column</div>
      <div class="column padding-10 bg-gray-light text-center">.column</div>
    </div>
  </div>
</div>

As seen here, if we nest two .columns inside of another .column, the width of the inner columns will be half of the outer one. For each level we nest, the parent element acts as the new "viewport" width to be split in 24 hgrid units.
