Installation

NPM or Yarn

From the root of your project, install hgrid-css as a dependency:

  $ npm install hgrid-css --save-dev

  $ yarn add hgrid-css --dev

Which additional packages you need in order to integrate hgrid as .sass/.scss in your project may depend on the framework you use. 

// React:
$ npm install sass --save-dev

// Vue.js:$ npm install sass sass-loader --save-dev

// Note that `node-sass` is deprecated as of October 2020. We use `sass` instead.

There are multiple ways to integrate hgrid in your projects, both as SCSS or with the pre-built css. Read a detailed description on how to set it up.

CDN 

  https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css

Include the link to the prebuilt hgrid kit from the CDN, or a local copy, before any other stylesheets in your html. You can customize hgrid by overwriting the CSS classes below the import in the code flow.

  <head>
  <link rel="stylesheet" href="https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css">
  <-- more css -->
</head>

You can also @import it at the top of your local stylesheet:

  /* From node_modules */@import './../etc./node_modules/hgrid-css/dist/hgrid.1.3.min.css';

/* From CDN */@import 'https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css';

Bake your own

Read more about how to integrate hgrid as sass in your projects.

  $ git clone https://github.com/ahansson/hgrid-css.git
