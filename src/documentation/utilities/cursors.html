Customize the experience for users on desktop devices with these hgrid cursor utility classes triggered by the :hover state. The graphical style fully depends on the browser implementations.

.cursor-pointer or .pointer

Displays a pointer cursor on any HTML element that is being hovered over (on non-touch-devices).
