Width Utilities

The majority of the width and max-width utilities follow the naming pattern .[max-]width-[n]pct. The array of numbers replacing the n variable is like this by default: [100, 90, 80, 75, 70, 66.666666, 60, 50, 40, 33.333333, 30, 25, 20, 10, 5].

You can customize hgrid's CSS output by overriding the array $fractions from _variables.scss in your own scss entry point file. Read more about how to customize it.

Note: The class names omit the decimals in the numbers array, using the integer values only. The CSS property uses the exact floating point value internally, for example width: 66.666666%;

Examples:
  • .width-5pct
  • .width-10pct
  • .width-33pct
  • .max-width-90pct
  • .max-width-66pct
  • .max-width-50pct

In addition we have these utilities for widths:

Utility Name CSS Property
.width-auto width: auto;
.min-content width: min-content;
.max-content width: max-content;
.fit-content width: fit-content;
.fit-content-min min-width: fit-content;
.fit-content-max max-width: fit-content;
.min-content-min min-width: min-content;
.min-content-max max-width: min-content;
.max-content-min min-width: max-content;
.max-content-max max-width: max-content;

Height Utilities

Utility Name CSS Property
.height-auto height: auto;
.height-100pct height: 100%;
.height-100vh height: 100vh;
.min-height-100 min-height: 100%;
.min-height-100vh min-height: 100vh;
