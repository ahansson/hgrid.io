Below we will look at some examples on how you can set up hgrid to work with your new or existing projects, and how to customize the default settings.

Vue 3

1. Install two dependencies: npm i sass sass-loader --save-dev (node-sass is deprecated)

2. Install hgrid-css: npm install hgrid-css

3. Create main.scss example in assets/scss

4. Initiate main.scss in your main.js file, respecting your local folder structure: 

import './assets/scss/main.scss'
