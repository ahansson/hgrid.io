Hgrid comes with two types of variables. Sass $variables for internal use and customization with Sass (SCSS), and regular CSS --custom-properties accessible from anywhere in your project.

Sass variables are configurable with the @use rule if you import hgrid as Sass into your project. Example: 

@use 'hgrid-css/sass/hgrid' with (
  $theme-color-primary: #ffe88d,
  $theme-color-secondary: #fff0b4,
  $link-underline-color: #ffe88d,
  $container-max-width: 1680px,
  $custom-stack-point: 572px
);

This overwrites the default values and replaces them with your custom values. Notice the use of commas, not semicolons.

See also: How to integrate hgrid in your Node-driven projects

Sass variables are abstracted away and converted to regular CSS during compilation. They aren't available in project components unless you explicitly import them into every file. To avoid this overhead, hgrid exposes the most useful internal variables as regular CSS custom properties, which are by default available everywhere via the global :root element.

As all the custom properties get their values from internal Sass variables, you can customize them with the @use configuration described above. Later in your CSS or SCSS or component files you can use them freely as is, overwrite them, or replace them with JavaScript.

Example of some of the custom properties available:

:root {

  // Breakpoints

  --phone: #{$phone};                        // 0 - 600px
  --phone-landscape: #{$phone-landscape};    // 601 - 768px
  --tablet-portrait: #{$tablet-portrait};    // 769 - 900px
  --tablet-landscape: #{$tablet-landscape};  // 901 - 1024px
  --laptop: #{$laptop};                      // 1025 - 1280px
  --desktop: #{$desktop};                    // 1281 - 1680px
  --desktop-large: #{$desktop-large};        // 1681 - ∞

  // Theme Colors

  --theme-color-primary: #{$theme-color-primary};
  --theme-color-secondary: #{$theme-color-secondary};
  --theme-color-tertiary: #{$theme-color-tertiary};
}

You can now (for example) use variables to write @media queries:

@media (min-width: var(--phone-landscape)) {
  .example {
    color: var(--theme-color-secondary);
  }
}

Another example use case: Custom color themes for your site. With a data property you can use JavaScript to replace the initial colors with an alternate set of variables: 

[data-theme="dark"] {
  --theme-color-primary: #ffc9a2;
  --theme-color-secondary: #d09255;
  --theme-color-tertiary: #d09255;
  --bg: #060323;
}

JavaScript:

const toggleColorTheme = () => {
    if (html.dataset.theme === 'light') {
      document.documentElement.setAttribute('data-theme', 'dark')
      localStorage.theme = 'dark'
    } else {
      document.documentElement.setAttribute('data-theme', 'light')
      localStorage.theme = 'light'
    }
  }
