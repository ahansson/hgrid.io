Headlines

Heading elements h1 to h6 are styled out of the box.

Hello, this is h1

Hello, this is h2

Hello, this is h3

Hello, this is h4

Hello, this is h5
Hello, this is h6

You can achieve the same visual result by using the utility classes .h1, .h2 and so on, with any HTML element. This lets you for example keep your markup semantic while at the same time reducing or increasing the font size relative to the element's default font size: 

<h2 class="h4">An h2 headline, with a smaller font size</h2>

Text columns

Split a text container into responsive columns with .two-columns, .three-columns or .four-columns on the parent element. 

  <div class="two-columns padding-40 border-gray-light radius-10 ">
  <p>Lorem ipsum dolor ...</p>
  <p>Curabitur sodales ...</p>
  <p>Class aptent taciti...</p>
</div>
