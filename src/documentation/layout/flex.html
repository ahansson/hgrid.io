Flex is perfectly suited for utility classes. Without writing a single line of CSS you'll be able to create any kind of flex based layout.

File location: sass/layout/_flex.scss

Code example
<div class="flex f-wrap" style="height:20rem; color:#fff;">
  <div class="f-grow-1 bg-gray">
    <div class="bg-gray-darker f-center padding-20">Content</div>
  </div>
    <div class="f-center f-grow-1 bg-gray-dark">Content</div>
    <div class="f-center f-4 bg-gray-darker">Content</div>
</div>
