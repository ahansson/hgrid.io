Fixed column widths

Add class="layout-fixed" to the table element, and every column will get the same width. Or style the first table cells with CSS in order to control each column width independently.

Adding .striped helps distinguish the rows from each other.

Scrollable tables

By wrapping the table in an HTML element with a limited width and a class of .scrollable-x, the table will be scrollable horizontally, given that its content overflows the container width. Only tables cells, rows or columns with a max-width or a width, will at some point become bigger than the available viewport width. For vertical scrolling, use an element with .scrollable-y and a fixed height.
