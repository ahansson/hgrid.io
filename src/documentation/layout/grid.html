CSS grid is an optional hgrid module that allows you to create complex, bi-dimensional layouts.

File location: sass/layout/_grid.scss

Setup

  • The grid utilities can be switched on and off with Sass @use or directly in _variables.scss, by modifying the $include-grid variable (default: true).
  • Responsive grid utilities can be excluded by changing $include-grid-responsiveness to false.
  • Max columns per grid parent is controlled with $grid-columns (default: 12).
  • Max rows per grid parent is controlled with $grid-rows (default: 6).
  • Pre-compiled CSS: To keep the file size low, CSS grid is excluded from hgrid.min.css. If you want the grid utilities included, use dist/hgrid.grid.min.css instead.

Note: Turning off CSS grid utilities is redundant if you plan to purge your CSS with the proposed build process.

Configuration Example with Sass

// initialize hgrid from node_modules with optional settings

@use 'hgrid-css/sass/hgrid' with (
  $include-grid: true,
  $include-grid-responsiveness: false,
  $grid-columns: 14,
  $grid-rows: 8
);

Basic Grid Example

Let's make a regular div a grid container with .grid. It's now the grid parent. 

<div class="grid">
  <p>Grid Child</p>
  <p>Grid Child</p>
</div>
Grid Row and Grid Column

Let's add 3 columns (horizontal) and 2 rows (vertical). It doesn't matter how many child elements we have. Invisible grid "tracks" will be created no matter what. 

<div class="grid g-column-3 g-row-2">
  <p>Grid Child</p>
  <p>Grid Child</p>
</div>

Gap

We need some spacing between our grid children. Let's add .gap-5 to the parent. It corresponds to a gap size of 1vw. Gap utilities are counted from .gap-1 to .gap-19 with both fixed (px and rem) and flexible (vw) values.

.gap-* apply equal spacing on all sides of child elements. If you want a different gap between columns and rows, use .column-gap-* and .row-gap-* respectively. 

// Array of included gap widths

$gaps:
  1px,
  0.4vw, 0.6vw, 0.8vw, 1vw, 1.6vw, 2vw, 2.6vw, 3vw, 
  0.2rem, 0.4rem, 0.6rem, 0.8rem, 1rem, 1.2rem, 1.4rem, 1.6rem, 1.8rem, 2rem;


<div class="grid g-column-3 g-row-2 gap-5">
  <p class="text-center mb-0 padding-20">1</p>
  <p class="text-center mb-0 padding-20">2</p>
  <p class="text-center mb-0 padding-20">3</p>
  <p class="text-center mb-0 padding-20">4</p>
  <p class="text-center mb-0 padding-20">5</p>
  <p class="text-center mb-0 padding-20">6</p>
</div>
