Push and pull

File location: sass/layout/helpers/_adjustment.scss

Push and pull classes are used to fine tune the position of html elements.

We use two methods: push and pull classes, or translate-push and translate-pull classes. The first will affect sibling element(s), while the latter will move the element independently of surrounding elements.

Class Name Comment
.push-1v (...) -20v Push 1-20px vertically (down)
.pull-1v (...) -20v Pull 1-20px vertically (up)
.push1h (...) -20h Push 1-20px horizontally (right)
.pull1h (...) -20h Pull 1-20px horizontally (left)
.translate-push-1v (...) -20v Move 1-20px down
.translate-pull-1v (...) -20v Move 1-20px up
.translate-push-1h (...) -20h Move 1-20px right
.translate-pull-1h (...) -20h Move 1-20px left
