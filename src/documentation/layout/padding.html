General Padding

The padding utilities have 5 and 10 pixels incrementations, between 0 and 150 pixels. They use rem units internally, but their naming reflect the pixel value they correspond to with the default hgrid configuration. Applying padding-80 to an elements gives it 8rem of visually adjusted padding on all sides which correspond to 80px.

  • padding-0
  • padding-5
  • padding-10
  • padding-20
  • padding-30
  • ...
  • padding-150

Individual Padding

  <div class="bg-gray pt-50 pr-90 pb-80 pl-40 min-content">
    <div class="bg-gray-dark border-gray-lighter border-dashed" style="height:5rem; width:20rem;"></div>
  </div>

Result:

Class Name CSS Property
.pt-0
.padding-top-0		 padding-top: 0;
.pt-10
.padding-top-10		 padding-top: 1rem;
.pt-20
.padding-top-20		 padding-top: 2rem;
etc. ...
.pt-140
.padding-top-140		 padding-top: 14rem;
.pt-150
.padding-top-150		 padding-top: 15rem;

Following the pattern above, you can add the class names .pr-* / .padding-right-*, .pb-* / .padding-bottom-* or .pl-* / .padding-left-*. Values from 0 to 150 incremented by 10.

Responsive Overrides

Let's revisit the example above and add a responsive modification of the padding-bottom. On screens smaller than tablets in landscape mode (tablet-landscape) we decide to remove the top padding completely: 

  <div class="bg-gray pt-50 pr-90 pb-80 pl-40 tablet-landscape-pt-0 min-content">
    <div class="bg-gray-dark border-gray-lighter border-dashed" style="height:5rem; width:20rem;"></div>
  </div>

The result can now be seen if you scale your viewport down to less than 1024 pixels:

Use the override utilities along this pattern:

  • .desktop-large-p-10
  • .desktop-large-pt-10
  • .desktop-large-pr-10
  • .desktop-large-pb-10
  • .desktop-large-pl-10
  • .phone-landscape-p-100
  • .phone-landscape-pt-100
  • .phone-landscape-pr-100
  • .phone-landscape-pb-100
  • .phone-landscape-pl-100

The default viewport laptop (<1280px) is excluded from the list of viewports you can use to construct these overrides.

These are your alternatives: .desktop-large (>1680px), .desktop (<1680px), .tablet-landscape (<1024px), .tablet-portrait (<900px), .phone-landscape (<768px) and .phone (<600px).
