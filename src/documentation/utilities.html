Complete collection of utilities can be found in grid > _utilities.scss. Margin-bottom classes are located in overrides > _overrides.scss. Examples:

  • .padding-0, -10, -20, etc.
  • .padding-top-0, -10, -20, etc.
  • .padding-bottom-0, -10, -20, etc.
  • .padding-left-x, .padding-right-x
  • .hidden, .visible
  • .pointer, .inline-block,
  • .absolute-center, .align-items-center, .text-center
  • float right (.fr), float left (.fl), float none (.fn)
  • .align-items-center (creates flex element)

1. Layout adjustment

Push and pull classes can be used to fine tune the position of html elements. Two methods: margin or transform. The first will also affect the subsequent element(s), while the latter will move its element independently of neighbors.
