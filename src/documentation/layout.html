There are various ways you can create your layout system with hgrid. Use the provided flex or grid classes, or whip up something super fast and precise with the built-in, automatic content columns.

Containers

File location: sass/layout/_containers.scss

Keep your content from falling off to the sides by wrapping it in a .container-max. This lets you control the max-width of your page with the scss variable $container-max-width (or you can overwrite .container-max in your regular css).

Hgrid's 1440 pixel .container-max on a 1680 pixel screen.

The default .container-max width is set to 1440 pixels.

Use .container if you instead want to use the full viewport width.

In order to group content in vertical sections within a container, use the class .row (creates a flex container) or .row-block (creates a container for block elements).

Creating layouts with the automatic content columns requires the use of .row.

.row-block is made for full width content that don't need columns, like headlines, text with floating images, etc.

.group can be used to group rows of content in sections.
