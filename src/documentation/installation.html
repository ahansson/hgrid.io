NPM or Yarn

From the root of your project, install hgrid-css as a dependency:

  $ npm install hgrid-css --save-dev

  $ yarn add hgrid-css --dev

Which additional packages you need in order to integrate hgrid as .sass/.scss in your project may depend on the framework you use. 

// React:
$ npm install sass --save-dev

// Vue.js:$ npm install sass sass-loader --save-dev

// Note that `node-sass` is deprecated as of October 2020. We use `sass` instead.

There are multiple ways to integrate hgrid in your projects, both as scss or with the pre-built css. Check the detailed description on how to set it up.

CDN 

  https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css

Include the link to the precompiled hgrid kit from the CDN, or a local copy, before any other stylesheets in your html. You can customize hgrid by overwriting the CSS classes below the import in the code flow.

  <head>
  <link rel="stylesheet" href="https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css">
  <-- more css -->
</head>

You can also @import it at the top of your local stylesheet:

  /* From node_modules */@import './../etc./node_modules/hgrid-css/dist/hgrid.1.3.min.css';

/* From CDN */@import 'https://unpkg.com/hgrid@1.3.0/dist/hgrid.1.3.min.css';

Bake your own

Note: If your project already has a build system with Sass compilation and CSS processing in place, read this instead: How to integrate hgrid as an npm module in your project. 

  $ git clone https://github.com/ahansson/hgrid-css.git

With the source code you can modify or extend hgrid to fit your needs. From the root of the hgrid-css folder, run npm install. This will set up the dev dependencies needed to run the Sass compiler and build the final CSS files.

With Node.js installed, use npm run watch to listen for changes and compile the result as CSS in the dist/ directory.

Add to, remove from, or edit the entry file sass/hgrid.scss in order to modify the final build.

We use autoprefixer to add some extra backwards browser compatibility and minify the final result with PostCSS. Use npm run build to prefix, minify and output three finished css files to the dist/ directory:

  • hgrid.css (prefixed, uncompressed
  • hgrid.min.css (prefixed, minified)
  • hgrid.css.map (source map for hgrid.min.css)

Adding your own custom style partials to the sass/ directory is the quick and dirty way to reuse and modify hgrid's variables while taking advantage of the existing compilation tools. If you wish to keep things more separated, which is recommended, please refer to the integration article for pointers.
